# PaVa-V3

A clean-room messaging project built for one goal: a lightning-fast messenger.

Workspace
- b3: backend
- f3: web frontend
- a3: Android app
- docs: architecture and contracts

Ground rules
- No code carryover from older versions
- New database
- New Firebase project
- Shared API and realtime contract across web and Android
- Speed before feature breadth
